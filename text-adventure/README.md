# Text Adventure Discord Bot

A specialized version of discord-dreams focused on text adventure game mastering.

## Overview

This bot acts as an AI Game Master for text adventures, providing:
- Rich narrative descriptions
- Scene-based image generation
- Interactive storytelling
- Player choice management

## Core Components

- `adventure.py`: Main bot implementation
- `launcher.py`: GUI for bot management
- `config.py`: Configuration settings
- `gamemasters.py`: Game Master definitions and prompts
- `scene_manager.py`: Scene-based image generation
- `adventure_manager.py`: Game state and conversation management

## Setup

1. Configure environment variables in `.env`
2. Configure game masters in `gamemasters.py`
3. Launch using `python launcher.py`

## Commands

- `!pic` - Generate an image of the current scene
